# Flappy Bird Clone 🐦

A simple *Flappy Bird* clone built using Pygame. The game features a bird that the player controls by pressing the spacebar to jump and avoid oncoming pipes. The pipes move from right to left, and the player scores points by successfully passing through them. 

## Features 🚀

- **Countdown before starting**: The game begins with a 3-second countdown ⏳.
- **Score System**: Points are awarded each time the bird passes through a pipe 🎯.
- **Increasing Difficulty**: The pipe speed increases every 5 points ⚡.
- **Collision Detection**: The game ends when the bird collides with a pipe or goes out of bounds 💥.
- **Game Over Screen**: Displays the final score and options to restart or quit 🎮.
- **Scrolling Background**: A smooth background scrolls horizontally 🌅.
- **Smooth Gameplay**: The game runs at 60 FPS for a fluid experience 🎥.

## Controls 🎮

- **Spacebar**: Makes the bird jump ⬆️.

## Requirements 📦

- Python 3.x 🐍
- Pygame library 🎨
